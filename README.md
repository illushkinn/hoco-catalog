# 📱 Hoco Catalog

Catálogo web para venta mayorista de accesorios HOCO en Mar del Plata, Argentina.

## 🚀 Quick Start

```bash
# Instalar dependencias
pnpm install

# Desarrollo
pnpm dev

# Build
pnpm build
```
