OPCION 1: Automatica (RECOMENDADA)
-----------------------------------
Ejecuta: LIMPIAR_Y_PUSH.bat

Esto hara:
1. Eliminar el historial de Git (.git)
2. Crear un nuevo repositorio limpio
3. Agregar .gitignore para excluir .project-config.json
4. Hacer commit inicial SIN credenciales
5. Push forzado a GitHub

NOTA: Perderas el historial de commits, pero es la forma mas rapida.

OPCION 2: Manual (Conserva historial)
--------------------------------------
Usa git-filter-repo para limpiar archivos especificos:

1. Instala git-filter-repo:
   pip install git-filter-repo

2. Limpia los archivos:
   git filter-repo --path .project-config.json --invert-paths
   git filter-repo --path client/.project-config.json --invert-paths

3. Push forzado:
   git remote add origin https://github.com/illushkinn/hoco-catalog.git
   git push -u origin main --force
