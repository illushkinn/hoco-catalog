### OPCIÓN 3: Usar CSS para Simular Transparencia (Temporal)

Mientras procesas las imágenes, puedo aplicar CSS que simule mejor la transparencia:

```css
.product-image {
  background: white;
  mix-blend-mode: multiply;
  /* Esto hace que los fondos blancos se vean transparentes */
}
```
